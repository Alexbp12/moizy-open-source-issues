# Markdown Previewer

A lightweight JavaScript Markdown Previewer that converts Markdown syntax into formatted HTML in real time.

---

## Features

- Live markdown preview
- Headings support
- Bold formatting
- Italic formatting
- Nested formatting support
- Lists
- Links
- Code blocks
- Blockquotes
- Horizontal rules
- Responsive layout
- Safe HTML escaping

## Supported Markdown Syntax

### Headings

```markdown
# Heading 1
## Heading 2
### Heading 3
```

### Bold

```markdown
**bold text**
```

### Italic

```markdown
*italic text*
```

### Links

```markdown
[GitHub](https://github.com)
```

### Lists

```markdown
- Item 1
- Item 2
```

### Code Blocks

````markdown
```js
console.log("Hello");
```
````

### Blockquotes

```markdown
> This is a blockquote
```

### Horizontal Rules

```markdown
---
```

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

---

## How It Works

The application listens for user input inside the markdown editor and converts markdown syntax into formatted HTML using JavaScript parsing logic and regular expressions.

The preview updates automatically in real time while typing.
